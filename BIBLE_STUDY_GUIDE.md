## üì± INSTALLATION ON YOUR PHONE

### iPhone/iPad (Safari):

1. **Open the app:**
   - Transfer `bible-index.html` to your phone (email, AirDrop, cloud storage)
   - Open it in Safari browser
   - OR host it online first (see GitHub guide below)

2. **Add to Home Screen:**

## üåê HOSTING ON GITHUB PAGES (RECOMMENDED)

Hosting makes installation much easier and allows sharing with others!

### Step-by-Step GitHub Hosting:

1. **Create GitHub Account**
   - Go to https://github.com
   - Sign up for free (if you don't have an account)

2. **Create New Repository**
   - Click **"+"** ‚Üí **"New repository"**
   - Repository name: `kingdom-daily-bible-study`
   - Description: "365 Days of In-Depth Bible Study"
   - Select **"Public"**
   - Check **"Add a README file"**
   - Click **"Create repository"**

3. **Upload Your Files**
   - Click **"Add file"** ‚Üí **"Upload files"**
   - Upload these 3 files:
     * `bible-index.html`
     * `bible-manifest.json`
     * `bible-service-worker.js`
   - Commit message: "Add Kingdom Daily Bible Study app"
   - Click **"Commit changes"**

4. **Rename Main File**
   - Click on `bible-index.html` in your repository
   - Click the **pencil icon** (edit)
   - Change filename to: `index.html`
   - Click **"Commit changes"** twice

5. **Enable GitHub Pages**
   - Go to **Settings** tab
   - Click **"Pages"** in left sidebar
   - Under "Source":
     * Branch: Select **"main"**
     * Folder: **"/ (root)"**
   - Click **"Save"**

6. **Get Your URL**
   - Wait 1-2 minutes
   - Refresh Settings ‚Üí Pages
   - Your app is live at:
   ```
   https://YOUR-USERNAME.github.io/kingdom-daily-bible-study/
   ```

7. **Share with Others!**
   - Anyone can visit this URL
   - They can install it on their phones
   - Perfect for church groups, small groups, families

## üîß TROUBLESHOOTING

**App won't install?**
- Ensure you're using Safari (iOS) or Chrome (Android)
- Check available storage space
- Try clearing browser cache first

**Can't see the content?**
- Make sure all 3 files are uploaded to GitHub
- Verify you renamed the file to `index.html`
- Wait 2-3 minutes after enabling GitHub Pages

**Navigation not working?**
- Clear browser cache
- Reinstall the app
- Check that JavaScript is enabled

**Want to update content?**
- Edit `index.html` on GitHub
- Changes go live in 1-2 minutes automatically
